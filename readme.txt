A simple Student Course Registration API

A Student who is not a team lead can 
1. Log in
2. Check the courses he registered for
3. Register for new courses


A Student who is a team_lead can
1. Log in
2. Check the courses he registered for
3. Register for new courses
4. Can appoint another student as team lead
5. Can delete a student
6. Can See a list of registered students
7. Can add a new student
8. Can get the details of a single student
